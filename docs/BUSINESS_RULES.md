# CHAVEIRO System

# Regras de Negócio

Este documento descreve todas as regras de funcionamento do sistema.

Nenhuma IA poderá alterar estas regras sem autorização.

---

# Empresas

Uma empresa possui:

- usuários
- clientes
- estoque
- ordens de serviço
- financeiro
- configurações

Todo dado pertence a apenas uma empresa.

---

# Usuários

Cada usuário possui:

- perfil
- permissões
- unidade
- empresa

Perfis:

Administrador

Gerente

Técnico

Atendente

Financeiro

---

# Clientes

Cliente pode possuir:

Pessoa Física

Pessoa Jurídica

Cada cliente pode possuir vários veículos.

---

# Veículos

Cada veículo possui:

Marca

Modelo

Ano

Placa

Chassi

Observações

# IA

A IA deve auxiliar apenas.

Nunca modificar dados automaticamente.

Sempre solicitar confirmação do usuário.
