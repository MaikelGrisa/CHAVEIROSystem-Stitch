# Financeiro

## Objetivo

O módulo Financeiro é responsável pelo gerenciamento completo das movimentações financeiras da empresa.

Todo lançamento financeiro deverá possuir rastreabilidade, auditoria e vínculo com os módulos operacionais quando aplicável.
