# CHAVEIRO System
## Guia Oficial da Interface

Este documento define todas as regras visuais do sistema.

Nenhuma IA poderá alterar estes padrões.

---

# Objetivo

Manter identidade visual consistente em todo o sistema.

---

# Estilo

Interface moderna.

Visual limpo.

Design profissional.

Layout empresarial.

Não utilizar aparência infantil.

---

# Componentes

Sempre reutilizar os componentes existentes.

Nunca criar componentes duplicados.

---

# Cores

Utilizar sempre a paleta oficial.

Não trocar cores principais.

Não alterar identidade visual.

---

# Espaçamentos

Todos os módulos devem manter:

- padding consistente
- margens iguais
- alinhamentos padronizados

---

# Tipografia

Utilizar apenas a fonte oficial.

Não alterar tamanhos sem necessidade.

---

# Botões

Mesmo padrão:

- bordas
- altura
- raio
- sombra
- animação

---

# Cards

Mesmo padrão em todas as telas.

Nunca criar estilos diferentes.

---

# Formulários

Campos sempre padronizados.

Mesmo espaçamento.

Mesmo comportamento.

---

# Tabelas

Mesmo layout.

Mesmas ações.

Mesma paginação.

Mesmo estilo visual.

---

# Responsividade

Desktop

Tablet

Mobile

Sempre suportados.

---

# Tema

Manter o tema atual do projeto.

Não alterar identidade visual sem autorização.
